# RC.3 问题反馈模板

每个独立问题先脱敏，再作为一行摘要加入 `rc-3-findings.md`。不确定是否敏感时，不附原始值。

## 严重级别

- **P0**：敏感数据泄露、数据丢失、操作错站或不可逆破坏。
- **P1**：主流程不可用、权限死循环、批量任务卡死，或导入导出无法恢复。
- **P2**：支持范围内稳定复现并明显影响工作流、正确性或可读性。
- **P3**：轻微视觉或便利性问题，有明确规避方式。

## 提交前检查

- [ ] 没有完整或可恢复的 Key、Cookie、Authorization header、账号标识或令牌编号。
- [ ] 没有完整存储、完整导出、恢复快照、浏览器 profile 或测试下载文件。
- [ ] URL 已去掉 query/hash；不需要域名时改用 `example.invalid`。
- [ ] 截图遮住地址栏、扩展 ID、通知、下载路径和无关私人内容。
- [ ] 诊断由“复制脱敏诊断”生成，并再次人工检查。

P0/P1 发现后立即停止使用当前 RC、撤销测试凭据并通知维护者。禁止以聊天记录代替问题台账。
