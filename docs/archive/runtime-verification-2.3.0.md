## 验收清单

- [ ] 在 `chrome://extensions` 或 `edge://extensions` 重新加载扩展，打开 service worker DevTools，确认 Console 无错误。
- [ ] 打开弹窗，确认窗口宽度、概览卡片、统计卡片和主操作按钮均比旧版更宽松；列表卡片不会把名称、域名、状态和操作挤在一行。
- [ ] 在未授权的新测试站执行「收藏当前页」或「刷新余额」，确认只请求该站 HTTPS 访问权限；拒绝后不执行网络、Cookie 或页面脚本操作。
- [ ] 在已登录测试站执行收藏和自动获取 Key，确认可正常识别、保存，且不会把 Key 发送到外部服务。
- [ ] 在正常站点上确认两次点击内可分别完成复制调用和复制 Key；前者只有 `https://域名/v1`，后者只有 Key。
- [ ] 添加多个标签，确认标签在卡片、设置页列表和搜索结果中可见；导出再导入后标签保留。
- [ ] 在余额批量刷新中重新加载扩展或关闭浏览器；重新打开后确认显示「余额刷新已中断」，再次点击刷新只继续剩余站点。
- [ ] 执行「本扩展格式」导出，确认下载前出现包含完整 API Key 的安全警告。
