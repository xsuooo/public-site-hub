## 验收清单

### 基础与布局

- [ ] 重新加载扩展后 SW Console 无错误。
- [ ] 弹窗宽度/卡片布局宽松，列表可滚动，主按钮可用。

### 权限（单站）

- [ ] 未授权站点「收藏当前页」或「查余额」或「重新识别」时，**只**请求该站 `https://域名/*`。
- [ ] **拒绝**授权后：不执行网络/Cookie/注入；toast 出现，可点 **「重试并授权」** 再次弹出权限。
- [ ] **允许**后同一操作可完成。

### Key

- [ ] 已有完整 Key：点「自动获取 Key」→ 提示已有/导入，**无**创建确认框。
- [ ] NewAPI 且令牌列表为空：点自动获取 → 出现「永不过期、无限额度」确认 → **取消**则不创建；**确定**则创建并保存。
- [ ] 失败为「请先打开令牌页」类时：toast 旁有 **「打开令牌页」**，点击能打开对应页。
- [ ] 非 NewAPI 无 Key 卡片按钮文案为 **「从页面导入 Key」**（或等价）；NewAPI/未识别为「自动获取 Key」。

### 复制

- [ ] 「复制调用」仅为 `https://域名/v1`，不含 Key。
- [ ] 「复制 Key」仅为 Key 本身。

### 批量余额

- [ ] 多站中部分从未授权：刷新余额时**不会**一次申请全部域名；已授权站先刷，未授权站逐个弹权。
- [ ] 对某一站点拒绝权限：该站记为跳过，其余站继续；完成摘要含「未授权跳过」或进度中的跳过数。
- [ ] 刷新中途「重新加载」扩展 → 再开弹窗显示「余额刷新已中断」→ 再点刷新只继续剩余。

### 余额失败分级（2.3.3）

- [ ] 未登录站查余额失败后：卡片红条显示分级文案，并有 **「打开站点」** + **「再查余额」**。
- [ ] 类型错误（如 404 类）失败后：有 **「重新识别」** 入口。
- [ ] 故意拖慢/断网时：超时失败文案可读，不卡住整批刷新（其它站可继续）。

### 其它

- [ ] 多标签在卡片/设置/搜索可见；本扩展格式导出导入后标签保留。
- [ ] 「本扩展格式」导出前有完整 API Key 风险警告。
- [ ] 签到相关：弹窗无签到同步入口；不会静默推送到签到扩展。
